🚗 UniGo - CampusPool
UniGo is a premium, glassmorphism-inspired carpooling platform exclusively designed for college students in Bangalore. Minimize your carbon footprint, split travel costs, and build a safer campus community through peer-to-peer ride sharing.

✨ Key Features
🛡️ Student-Only Verification: Compulsory college email verification (e.g., @newhorizonindia.edu) ensures a safe, exclusive environment.
🗺️ Live Map Integration: Powered by TomTom Maps SDK, featuring real-time location tracking and precise distance calculations.
🌓 Dual Mode: Seamlessly switch between Rider and Driver dashboards with a single tap.
💬 Automated Chat: Instant communication with automated "on-the-way" messages when a ride is accepted.
💳 Subscription System: Multiple tiers (Basic, Pro, Elite) featuring eco-score tracking and student rewards.
🎨 Premium UI: Modern dark mode with glassmorphism effects, fluid animations (React Native Animated API), and a high-end aesthetic.
🛠️ Tech Stack
Frontend: React Native (Expo), TypeScript, Zustand (State Management).
Backend: Python (FastAPI), Uvicorn.
Database: MongoDB (Async Motor Driver).
Maps: TomTom Maps SDK.
