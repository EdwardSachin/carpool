# 🚗 UniGo - CampusPool

**UniGo** is a premium, glassmorphism-inspired carpooling platform exclusively designed for college students in Bangalore. Minimize your carbon footprint, split travel costs, and build a safer campus community.

## ✨ Key Features

- 🛡️ **Student-Only Verification**  
  Compulsory college email verification (`@newhorizonindia.edu`) ensures a safe, peer-to-peer environment.

- 🗺️ **Live Map Integration**  
  Powered by **TomTom Maps SDK**, featuring real-time driver tracking and accurate distance calculations.

- 🌓 **Dual Mode**  
  Seamlessly switch between **Rider** and **Driver** dashboards.

- 💬 **Automated Chat**  
  Instant communication with automated ride-acceptance and “on-the-way” notifications.

- 💳 **Subscription System**  
  Multiple tiers (Basic, Pro, Elite) with eco-score tracking and student rewards.

- 🎨 **Premium UI**  
  Modern dark mode with glassmorphism, smooth animations (React Native Animated API), and a clean design.

## 🛠️ Tech Stack

- **Frontend**: React Native (Expo), TypeScript, Zustand  
- **Backend**: Python (FastAPI), Uvicorn  
- **Database**: MongoDB (Motor Async Driver)  
- **Maps**: TomTom Maps SDK
